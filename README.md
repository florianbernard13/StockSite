<section>
    <h2>Description</h2>
    <p>Ce projet permet de récupérer des données boursières et de les analyser. Il se concentre sur la récupération dynamique des données financières, l'affichage et l'analyse via des outils dédiés.</p>
</section>

<section>
    <h2>Prérequis</h2>
    <ul>
        <li>Python 3.x</li>
        <li>Flask</li>
        <li>yFinance</li>
        <li>Pandas</li>
    </ul>
</section>

<section>
    <h2>Installation</h2>
    <p>Clonez le projet sur votre machine.</p>
    <pre><code>git clone https://github.com/florianbernard13/StockSite.git</code></pre>
    <p>Créez un environnement virtuel :</p>
    <pre><code>python3 -m venv venv
source venv/bin/activate  # Sur Windows : venv\Scripts\activate</code></pre>
    <p>Installez les dépendances :</p>
    <pre><code>pip install -r requirements.txt</code></pre>
</section>

<section>
    <h2>Lancer le serveur</h2>
    <p>Pour démarrer l'application, exécutez la commande suivante :</p>
    <pre><code>python3 app.py</code></pre>
    <p>Le serveur sera accessible sur <a href="http://127.0.0.1:5000/">http://127.0.0.1:5000/</a>.</p>
</section>
